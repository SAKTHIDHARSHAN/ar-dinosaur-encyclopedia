# AR Dinosaur Encyclopedia

This is an Augmented Reality application built using Unity and AR Foundation.  
It lets users place dinosaurs in the real world and view basic information about them.

## Features

- AR plane detection using AR Foundation  
- Place dinosaur models in real environment  
- Simple UI to display dinosaur information  
- Multiple dinosaur selection  

## Technologies Used

- Unity  
- C#  
- AR Foundation  
- ARCore  
- TextMeshPro  

## What I Learned

- Basics of Augmented Reality in Unity  
- Plane detection and object placement  
- UI handling in Unity  
- Working with prefabs and scene management  

## How to Run

1. Open the project in Unity  
2. Build for Android (ARCore supported device required)  
3. Run the application
